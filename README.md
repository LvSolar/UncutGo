# UncutGo

UncutGo 是一个电影版本判断工具。

输入电影名或豆瓣链接后，它会：
- 搜索豆瓣候选电影
- 解析豆瓣版本与片长信息
- 抓取国内平台片长
- 判断哪些平台更可能接近未删减版本

## 当前能力

- 豆瓣实时搜索候选电影
- 候选只有 1 部时自动开始分析
- 电影基础信息展示：标题、年份、导演、评分、剧情简介
- 版本依据展示：当前基准版本 + 其他已识别版本
- 平台片长抓取：腾讯视频、哔哩哔哩、爱奇艺、优酷、Libvio
- 结果标签：大概率无删减 / 可能无删减 / 版本不一致 / 可能删减 / 疑似删减 / 信息不足
- 流式分析返回，平台结果逐步展示
- 最近搜索、重新抓取、停止分析

## 版本选择规则

当前基准版本优先级：
1. 导演剪辑版
2. 电影节版本
3. 国际版本
4. 加长版 / 修复版
5. 中国大陆版
6. 其他未知版本

同优先级下，优先选择更长版本。

这意味着如果同一部电影同时存在中国大陆版和电影节版，系统会优先拿电影节版做平台对比。

## 速度优化

目前已经做的优化：
- 预览、普通分析、流式分析共用豆瓣详情解析缓存
- 豆瓣挑战 Cookie 短时复用，减少重复过挑战页
- 候选悬停时只预拉轻量预览，不再偷偷触发整套重分析
- 分析请求可取消，可重新抓取

## 测试建议

建议重点测试这几类电影：
- 只有 1 个候选的电影：应自动开始分析
- 有多个版本的电影：如《心迷宫》《蝴蝶效应》
- 没有国内片源的电影
- Libvio 较慢或失败的电影
- 点击“重新抓取”后的结果刷新
