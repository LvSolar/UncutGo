# UncutGo

输入电影名或豆瓣链接，判断国内哪些播放源更可能是无删减版。

## 当前进度

当前版本已经接通：

- 真实豆瓣搜索
- 真实豆瓣详情页解析
- 豆瓣播放源识别
- 腾讯视频真实时长抓取
- 哔哩哔哩真实时长抓取
- 爱奇艺真实时长抓取
- 优酷真实时长抓取
- Libvio 站内搜索与播放页时长抓取
- 多版本参考规则与删减判断
- 无国内播放源的正式结果展示
- 本地文件缓存
- 结果页状态区分与说明文案优化

## 当前能力

目前已经支持：

- 输入电影名后返回豆瓣候选电影
- 输入豆瓣链接后直接进入分析流程
- 进入真实豆瓣电影详情页并解析导演、评分、简介、片长、播放源
- 展示多个已识别版本时长，并选出当前参考版本
- 对腾讯视频、哔哩哔哩、爱奇艺、优酷抓取真实页面或接口时长
- 对 Libvio 进行站内搜索、播放页定位与时长提取
- 基于参考版本输出 `大概率无删减 / 可能无删减 / 版本不一致 / 可能删减 / 疑似删减 / 信息不足`
- 把“无国内播放源”当作正式结果，而不是异常
- 返回实时结果或缓存结果
- 在结果页区分 `实时结果 / 缓存结果 / 样例数据 / 浏览器兜底 / 时长待确认 / 未搜到片源`

## 判定说明

当前判定遵循这些原则：

- 电影可能存在多个合法版本，结果里需要展示多个版本时长
- 参考版本优先级为：导演剪辑版 > 电影节版 > 非中国大陆版 > 其他国际版 > 中国大陆版
- 同优先级多个版本并存时，默认选更长版本作为参考
- 豆瓣分钟显示不等于精确秒数，不能把分钟取整误判为删减
- 平台比参考版本略长时，会优先容忍片头片尾、平台提示或黑场带来的常见差异
- 平台时长明显更长时，优先视为版本不一致，而不是无脑判成无删减
