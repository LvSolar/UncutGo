# FindMyMovie

输入电影名或豆瓣链接，判断国内哪些播放源更可能是无删减版。

当前版本已经接通：

- 真实豆瓣搜索
- 真实豆瓣详情页解析
- 豆瓣播放源识别
- 腾讯视频真实时长抓取
- 哔哩哔哩真实时长抓取
- 多版本参考规则与删减判断
- 无国内播放源的正式结果展示

## 当前能力

目前已经支持：

- 输入电影名后返回豆瓣候选电影
- 进入真实豆瓣电影详情页并解析导演、评分、简介、片长、播放源
- 对腾讯视频和哔哩哔哩抓取真实页面时长
- 基于参考版本输出 `大概率无删减 / 版本不一致 / 信息不足` 等标签

当前还没完成：

- 爱奇艺真实时长抓取
- 优酷真实时长抓取
- 更强的豆瓣搜索兜底策略
- 更细的版本标签提取
- 线上部署配置
